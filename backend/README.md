# MH Quiz Backend

Single Express API serving both public quiz and admin clients.

## Routes

- Public: `/api/quiz/*`
- Admin: `/api/admin/*`

## Quick start

1. Copy `.env.example` to `.env` and fill values.
2. Install dependencies.
3. Run migrations.
4. Start dev server.

## Scripts

- `npm run migrate`
- `npm run dev`
- `npm run check`
- `npm run build`
